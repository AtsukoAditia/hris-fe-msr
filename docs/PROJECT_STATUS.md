## Payroll Workspace

The Admin/HR payroll workspace at `/payroll` contains five responsive tabs:

- **Payroll:** list, filters, item detail, recalculation, review, finalization, paid, and cancellation.
- **Laporan:** server summary, period/status/employee filters, CSV/PDF export, and protected payslip download.
- **Periode:** payroll-period CRUD and draft generation.
- **Profil Gaji:** effective-dated salary and employee component assignments.
- **Komponen:** earning/deduction master administration.

## Employee Payslips

The employee-only `/payslips` route provides:

- finalized and paid payslip history;
- earning and deduction breakdown;
- attendance, absence, late, unpaid leave, and overtime metrics;
- pagination and empty/error/loading states;
- authenticated PDF download using Axios blob responses; and
- responsive tables on desktop and cards on mobile.

The frontend route guard is an experience control. Backend ownership and payroll-status authorization remain authoritative.

## Secure Downloads

`src/utils/downloadResponse.js`:

- reads the server-provided filename when available;
- creates a temporary object URL;
- triggers the browser download; and
- immediately revokes the object URL.

No token or document URL is placed in a query string or public storage path.

## Known Limitations

- Automatic PPh 21 and BPJS calculation remains out of scope.
- Post-finalization payroll adjustment remains a later milestone.
- PDF output is generated by the backend and intentionally uses a simple printable layout.
